# Urvi Shah - 2D Graphics Portfolio

Welcome to my professional portfolio website! This is a modern, responsive portfolio showcasing my work as a Manager in 2D Graphics at INVNT.

## 📋 Overview

This portfolio features:
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Professional background and key skills
- **Projects Section**: Showcase of featured graphics projects
- **Contact Section**: Multiple ways to get in touch
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile

## 🎨 Live Demo

View the live portfolio at: [https://urvishah231998.github.io/Urvi-Shah-Portfolio/]

## 🚀 Features

- Fully responsive design
- Smooth navigation with active link indicators
- Dynamic project loading from JavaScript
- Modern gradient designs and smooth animations
- Professional color scheme
- Integration with Behance portfolio link

### 3. Customize Colors

Edit `styles.css` and modify the CSS variables in the `:root` selector:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --dark-bg: #0f172a;
    --light-bg: #f8fafc;
    --text-primary: #1e293b;
    --text-secondary: #64748b;
    --border-color: #e2e8f0;
}
```
